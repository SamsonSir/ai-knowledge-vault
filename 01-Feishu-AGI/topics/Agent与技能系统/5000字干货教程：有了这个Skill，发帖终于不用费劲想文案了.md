### 📋 核心分析

**战略价值**: 用 Claude Code + social-post-writer Skill，把模糊想法一键转化为多平台社交文案，彻底消除"有想法但写不出来"的表达瓶颈。

**核心逻辑**:

- **表达瓶颈的本质不是没想法，而是从脑子到文字的转化卡壳**——把模糊感受丢给 AI，它能帮你说出你想说的话，你的反应往往是"对，这就是我想说的"
- **工具定位是"帮你说"而非"帮你想"**——你提供模糊想法/推广内容/参考帖子，它输出完整文案，核心价值是降低表达门槛
- **支持三种核心使用场景**：① 模糊想法→成型帖子；② 直接写推广文案；③ 基于别人帖子改写成自己的话
- **支持个人风格学习**：把自己过去几十条随手记、朋友圈、即刻动态喂给它，它能学习你的用词习惯、句式节奏、调性（不说教但有观点），实现"克隆表达风格"
- **多平台一次性适配**：单次调用同时生成推特/X（280字限制）、即刻（1000字限制）、微博（2000字限制）、小红书等多平台版本，每版本附字数统计和📋复制版
- **Skill 安装有两种方式**：① 通过 Claude Code 自然语言指令安装；② 手动复制文件夹到 `~/.claude/skills/` 目录，两者等效
- **Skill 来源平台是技能范（www.skills.fan）**，支持按标题或技能ID搜索，该 Skill ID 为 `000012`，名称为"社交媒体帖子"
- **输出格式标准化**：每个平台版本包含正文、字数/上限校验（如 `62 / 280 ✓`）、📋复制版三段，末尾附追问引导（风格/平台/个人经历）
- **AI时代核心竞争力重构**：执行力被AI拉平后，真正稀缺的是"会提需求"和"有值得表达的思考"，而非表达技巧本身
- **降低门槛比提升能力更重要**——先动手发出去，比纠结"我不会写"更有价值

### 🛠️ 操作流程

**1. 准备阶段：安装 Claude Code**
- 安装 Claude Code 并确保能正常调用 AI 模型（国内/国外模型均可）
- 参考橘子老师（公众号：有机大橘子）的入门指南：《开源模型质变：Claude Code 超级小白入门指南》

**2. 下载 Skill**
- 浏览器打开 `www.skills.fan`
- 点击顶部导航 → 「技能」进入技能页
- 搜索 `社交媒体帖子` 或技能ID `000012`
- 进入详情页 → 点击右上角「下载」
- 解压后得到文件夹：`social-post-writer`

**3. 安装 Skill（二选一）**

| 方式 | 操作 |
|------|------|
| 方式A：自然语言安装 | 打开终端启动 Claude Code，输入：`帮我安装电脑下载目录里的 social-post-writer 技能`，中途询问点 Yes |
| 方式B：手动安装 | 将 `social-post-writer` 文件夹复制到 `/Users/你的用户名/.claude/skills/` |

**4. 核心执行：三种使用场景**

直接在 Claude Code 中用自然语言描述需求，Skill 自动加载（`Successfully loaded skill`）并输出多平台版本。

### 📝 避坑指南

- ⚠️ 安装时 Claude Code 中途会有询问弹窗，必须点 Yes 才能完成安装，不要忽略
- ⚠️ 手动安装路径注意替换用户名：`/Users/【你的用户名】/.claude/skills/`，不是照抄 `zhuoyu`
- ⚠️ 风格学习功能需要主动喂数据：把自己过去的随手记/朋友圈/即刻动态提供给它，不会自动抓取
- ⚠️ 每次输出末尾的追问引导（风格/平台/个人经历）是优化入口，第一版不满意直接追问调整，不要重新开始
