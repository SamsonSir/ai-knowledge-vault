### 📋 核心分析

**战略价值**: 一套可直接部署的工业级 AI 导演分镜 Skill，分基础版（~30条规则）与进阶版（~50条规则）两层注入，覆盖从剧本解读到逐镜头输出的完整制作流程。

**核心逻辑**:
- **两版分层设计**：基础版保证单次推理高执行率，进阶版作为第二轮精修补充注入，两版互补不冲突，进阶版在重叠处优先级更高
- **镜头数量基准**：1-2分钟成片 → 60-120个镜头；500字剧本 ≈ 1分半时长；动作/战斗密集型偏上限，对话/氛围型偏下限，数量是规则自然落地的结果，不凑数不偷懒
- **最小拆解单位是"情绪节拍"**，严禁以笼统"场景"为单位，所有情绪转折点、信息爆发点、氛围转换点必须标记
- **前置建构强制要求**：开场增补5-15个无台词视觉引入镜头，第一镜定义世界观基调，默认避免平庸全景开场，优先局部特写引入再拉开全貌，最后一镜必须是强烈悬念钩子
- **动作四镜头链**：起势→过程→落点→反应，缺一不可；"落点"必须含物理反馈（灰尘/震动/碎裂）
- **对话拆镜默认规则**：单句台词 = 说话者起势→听者反应→说话者高潮→结束后反应；台词说完后留半秒再切走；听者反应优先级高于说者话语
- **衔接逻辑强制标注**：场景/情绪跳转处必须标注【主逻辑+次逻辑】复合标签；连续自然镜头允许单标签；善用J-Cut（声音先行）/L-Cut（画面先行）增强情感连接
- **视觉叙事底线**：关掉声音能理解至少60%的故事和情感（动作/视觉叙事型目标80%，对话驱动型目标60%）
- **角色塑造铁律**：无台词无动作时角色必须处于"平行反应链"中，严禁呆立；每个主要角色至少一个眼神特写镜头；角色镜头语言设计从"这个角色是谁"出发，而非从性别/外貌出发
- **终极戒律**：一个镜头没有同时服务于"叙事、角色、情绪"中至少两者，即为废片；先做减法直到减去任何一个镜头都损害故事，再做加法为人物增加一个"灵魂镜头"

### 🎯 关键洞察

**加载方式三选一，按场景选择**：
- 日常生成 → 只加载基础版，单次推理执行率最高
- 精修阶段 → 先跑基础版，再追加进阶版做第二轮注入
- 全量一次性 → 两个文件同时加载，适合对品质要求极致且上下文窗口足够的场景

**分镜的核心价值观**：分镜师90%的价值体现在"增补镜头"的想象力上，而非复述剧本。剧本是藏宝图，分镜是挖宝藏。

|------|--------|---------|---------|---------|
| 基础版 | `director-storyboard-core.md` | ~30条 | 日常生成/首轮输出 | 单次推理高执行率保障 |
| 进阶版 | `director-storyboard-advanced.md` | ~50条 | 精修/第二轮注入 | 重叠处以进阶版为准 |
| 全量加载 | 两文件同时加载 | ~80条 | 一次性高品质输出 | 需确保上下文窗口足够 |

| 模块 | 关键规则 | 预期效果 | 注意事项/坑 |
|------|---------|---------|-----------|
| Module A: 角色塑造 | 标志性姿态镜头、病态微动作（反派）、群像多层反应链 | 角色立体感、记忆点强 | 魅力必须是动态的，一个微动作>十秒静止特写 |
| Module B: 镜头语言 | 荷兰角=内心不安；深焦=人物关系；浅焦=内心世界；垂直俯拍仅用于宿命感/无力感 | 镜头语言精准传递情绪 | 给物件特写=宣布"它很重要"，不可滥用 |
| Module C: 动作拆解 | 拔剑=剑鞘特写→眼神特写→手握剑柄特写→拔出；打斗必须有慢动作展示力量顶点 | 动作戏节奏感强、有力度 | 受伤反应必须匹配角色性格，禁止套公式 |
| Module D: 对话拆镜 | 逗号=插入听者微表情；停顿=插入环境/细节镜头；台词说完后半秒再切 | 对话有呼吸感、情绪流畅 | 正反打角度/景别/焦距必须持续变化，禁止机械重复 |
| Module E: 衔接技法 | 匹配转场每场戏最多用一次；记忆闪回必须在色调/画幅/质感上与主时空明确区分 | 转场自然、节奏精准 | 手持/晃动镜头必须有情绪动机，禁止无理由使用 |

### 🛠️ 操作流程

**使用方式一：日常生成**
```
加载技能：director-storyboard-core.md
[粘贴剧本]
```

**使用方式二：精修阶段（两步走）**
```
# 第一步
加载技能：director-storyboard-core.md
[粘贴剧本]

# 第二步（首轮输出完成后）
追加技能：director-storyboard-advanced.md
请基于以上进阶规则，对已生成的分镜脚本进行精修。
```

**使用方式三：全量一次性加载**
```
加载技能：director-storyboard-core.md + director-storyboard-advanced.md
[粘贴剧本]
```

**执行流程（7步）**：
1. **剧本解读与节拍拆解** → 以情绪节拍为最小单位，标记所有转折点/爆发点/氛围转换点，判断题材类型
2. **前置建构** → 增补5-15个无台词视觉引入镜头，第一镜定基调，最后一镜必须是悬念钩子
3. **角色建立** → 每个主要角色增补专属人设镜头+标志性习惯动作+至少一个眼神特写
4. **逐镜头分镜创作** → 多景别/多视角/多元构图；动作四镜头链；对话拆镜处理
5. **衔接逻辑设计** → 每次切换标注衔接标签，跳转处用复合标签，善用J-Cut/L-Cut
6. **节奏控制** → 快节奏=密集卡点剪辑；慢节奏=延长单画面/长镜头
7. **质量自检** → 基础版6项自检 + 进阶版10项自检（共16项）

### 📦 输出格式规范

```markdown
| 镜头编号 | 关联剧本 | 景别/视角/构图 | 画面描述 | 情绪&节奏 | 声音设计 | 衔接逻辑 | 运镜提示 |
|---------|---------|--------------|---------|----------|---------|---------|---------|
| S001 | 原文引用或"增补镜头" | 大远景/低角度仰拍/黄金分割构图 | 详细画面内容：角色动作、表情、环境细节、光影、动态元素 | 压抑→紧张，慢节奏 | 环境音/音效/配乐/静默 | 【主逻辑+次逻辑】 | 缓推/横摇/固定 |
```

- 必须从头到尾完整输出，严禁省略/概括/缩写
- 因篇幅限制需分批时：每批30-40个镜头，批次末尾标注【续】，等待用户指令后继续
- 每批次保持镜头编号连续

### 📝 避坑指南

- ⚠️ 连续三个以上同景别镜头必须有明确节奏或情绪目的，否则视为无意识重复
- ⚠️ 硬切是合法剪辑语言，但必须有节奏或情绪动机，禁止无意识随意硬切
- ⚠️ 手持/晃动镜头必须有情绪动机，禁止无理由使用
- ⚠️ 给物件特写=宣布"它很重要"，不可滥用，滥用会稀释叙事重点
- ⚠️ 匹配转场每场戏最多用一次，过度使用会失去冲击力
- ⚠️ 垂直俯拍（上帝视角）仅在表达"宿命感"或"无力感"时使用，不可随意调用
- ⚠️ 空镜头默认控制在三秒内；氛围系/治愈系可适当延长，但画面内必须有持续视觉变化
- ⚠️ 长镜头（>10秒）结尾必须有"信息爆炸点"，否则观众注意力流失
- ⚠️ 记忆闪回必须在色调/画幅/质感上与主时空明确区分，否则观众混淆时间线
- ⚠️ 角色死亡的视觉符号设计不要只用"手中物体滑落"这一个方案，可以是光线熄灭、节奏停止、画面色彩流失等

### 🛠️ 16项完整自检清单

| # | 自检项 | 来源 |
|---|--------|------|
| 1 | 每个镜头是否服务于"叙事、角色、情绪"中至少两者？ | 核心版 |
| 2 | 是否存在无意识的连续同景别镜头？（刻意重复除外） | 核心版 |
| 3 | 场景/情绪跳转处的衔接逻辑是否均为复合标签？ | 核心版 |
| 4 | 对话镜头是否完成了合理的拆镜处理？（完整呈现处是否有明确创作动机？） | 核心版 |
| 5 | 镜头总数是否与题材节奏匹配？（非机械凑数，亦非偷懒缩减） | 核心版 |
| 6 | 关掉声音能否理解至少60%的故事和情感？ | 核心版 |
| 7 | 每个主要角色是否有基于其核心气质的"标志性姿态镜头"？ | 进阶版 |
| 8 | 反派登场是否伴随环境失衡？ | 进阶版 |
| 9 | 反派是否有病态微动作设计？ | 进阶版 |
| 10 | 群像反应是否拆解为多层反应链？ | 进阶版 |
| 11 | 正反打是否在角度/景别/焦距上持续变化？ | 进阶版 |
| 12 | 匹配转场是否控制在每场戏最多一次？ | 进阶版 |
| 13 | 最昂贵的镜头是否用在了情绪最高潮节点？ | 进阶版 |
| 14 | 角色的走路姿态/开门方式是否匹配其当前心态？ | 进阶版 |
| 15 | 是否有遗漏的潜台词表达机会（听者反应特写）？ | 进阶版 |
| 16 | 角色受伤/死亡的表现方式是否匹配其性格，而非套用公式？ | 进阶版 |
