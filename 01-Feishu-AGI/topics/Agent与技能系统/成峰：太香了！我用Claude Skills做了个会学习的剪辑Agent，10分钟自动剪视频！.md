### 📋 核心分析

**战略价值**: 用 Claude Code Skills 构建一个「自更新剪辑 Agent」，替代剪映的手工操作，实现 10 分钟自动剪 30 分钟口播视频，且越用越懂你。

**核心逻辑**:

- 剪映的根本缺陷是「无法理解语义」，导致重复片段漏识别、字幕错别字多，手工剪 20-30 分钟视频极耗时
- Agent 与剪映的本质区别：剪映 = 固定工具 + 手工操作；Agent = 自适应系统 + 自动学习，不是用更好算法替换，而是用「会自我完善的系统」替换
- Skills 拆分原则：不做一个大 skill 靠指令区分任务，而是把 5 个核心任务拆成 5 个独立 skill 放入 `.claude/skills/` 目录，原因是人需要在阶段性产出处做检查（如审查稿确认后再执行剪辑）
- 输入 `/v` 即可触发 Claude Code 列出全部 5 个可用技能菜单，选一个执行一个，无需重复说明方法论
- 自更新机制是核心竞争力：每次任务后给反馈，AI 把反馈永久写入 Skills 文件，用 10 次覆盖 80% 个人习惯，用 50 次完全符合个人需求
- 双模型分工：FunASR 负责字级别剪辑（识别口误、语气词），Whisper 负责字幕生成，两者各有优势不可互换
- 自定义词典是字幕准确率的关键杠杆：预置词典文件路径 `.claude/skills/videocut:字幕/词典.txt`，录入专有名词后 Whisper 转录时自动纠错，第一次就准确
- 审查稿格式有明确语义符号：⏸️ 表示静音片段（需删除），🔴～～ 表示重复片段（需删除），格式可按需让 AI 修改
- 整个工作流 4 步闭环：下载 Skills → 安装环境 → 剪口播（含审查+自更新） → 加字幕（含词典纠错+烧录）
- 最终效果与剪映手动剪辑结果基本一致，但全程自动化

### 🎯 关键洞察

**为什么「自更新」比「更好的算法」更有价值**：

传统工具的上限是固定的，再好的算法也只能逼近「通用最优」，无法逼近「你的最优」。自更新机制的逻辑链是：反馈 → 写入 Skills → 下次执行时直接应用 → 再反馈 → 再写入。每一次迭代都在缩小「通用规则」和「个人习惯」之间的 gap。这是一个复利结构，而不是线性改进。

**为什么要拆成多个独立 Skills 而不是一个大 Skill**：

人机协作的关键节点是「人的确认」。一个大 skill 会把所有步骤串成黑盒，人无法在中间介入。拆成独立 skill 后，每个 skill 是一个检查点，人可以在审查稿阶段发现问题并修正，避免错误传递到最终剪辑。

### 🛠️ 操作流程

**第一步：下载 Skills**

新建一个专用文件夹（作为剪辑工作空间），在 Claude Code 中输入：

```
帮我下载剪辑 Skills：
- 地址：https://github.com/Ceeon/videocut-skills
- 存放路径：.claude/skills/（直接克隆到此目录，不要创建子文件夹）
```

完成后重启 Claude Code，输入 `/v` 确认 5 个 skill 已出现。

**第二步：安装环境**

输入 `/v` → 选择 `videocut:安装`

AI 自动安装依赖并下载两个模型（FunASR + Whisper，合计约 5GB），等待完成即可。

**第三步：剪口播**

1. 输入 `/v` → 选择 `videocut:剪口播`
2. 告诉 AI 视频文件路径
3. AI 自动执行：转录视频 → 识别口误（逐句检查） → 识别语气词（嗯/哎/诶等） → 识别静音（≥1s） → 生成审查稿
4. 审查稿符号说明：⏸️ = 静音片段（删除），🔴～～ = 重复片段（删除）
5. 人工检查审查稿，如有不满意（如「保留更多的嗯」、「某词识别错了」）继续与 AI 沟通直到满意
6. 输入 `/v` → 选择 `videocut:自更新`，AI 将本次调整写入 Skills
7. 确认后 AI 自动执行剪辑

**第四步：加字幕**

1. 打开 `.claude/skills/videocut:字幕/词典.txt`，添加专有名词（公司名、产品名、频道名等），例如：
   ```
   成峰的频道
   AI产品自由
   我的品牌名
   ```
2. 输入 `/v` → 选择 `videocut:字幕`
3. AI 自动执行：Whisper 转录 → 词典纠正错别字 → 生成字幕文件
4. 确认字幕正确后，AI 自动烧录到视频
5. 如发现词典有遗漏，随时 `/v` → `videocut:自更新` 补充

---

### 📝 避坑指南

- ⚠️ Skills 克隆时必须直接放到 `.claude/skills/` 目录，不要创建子文件夹，否则 Claude Code 无法识别
- ⚠️ 安装环境需下载约 5GB 模型，提前确认磁盘空间充足
- ⚠️ 审查稿确认步骤不能跳过，这是人工介入纠错的唯一窗口，跳过会导致错误传递到最终视频
- ⚠️ 自更新必须主动触发（`videocut:自更新`），不会自动执行，不用就等于放弃了 Agent 的核心价值
- ⚠️ 词典文件要在第一次加字幕前录入，否则第一次转录就会出现专有名词错误
