### 📋 核心分析

**战略价值**: 一人闭环复刻 AI 动漫短剧的完整工业流水线，从小说原稿到可发布成品，8 个环节全部有对应 AI 工具和可直接复用的 Prompt。

**核心逻辑**:

- **环节1 — 小说选材**：选取现代校园类小说片段（叶辰苏清雪系列），只需开头一小段情节即可，无需完整小说，降低素材门槛。
- **环节2 — 剧本拆分**：将小说片段 + 编剧 Prompt 一起发给 DeepSeek（https://chat.deepseek.com/），AI 自动输出电影化剧本，格式包含：场景题头（加粗）、△开头的画面动作、紧凑对白格式（角色名加粗+情绪提示）、【音效】/【字幕】标注、(O.S.)/(V.O.) 专业术语，每集约 1 分钟时长。
- **环节3 — 分镜脚本**：将剧本 + 分镜导演 Prompt 发给 DeepSeek，输出标准分镜表格，列包含：镜号、时长（1-10秒）、景别（远/全/中/近/特写/大特写）、摄法（固定/横摇/推/拉/跟随等）、画面内容（≥50字，含视觉风格+环境+人物+动作+情绪+光影+构图+特效）、台词/音效、入镜角色、场景标识。
- **环节4 — 加入 AI 提示词列**：在分镜表格中额外增加一列"AI画面提示词"，后续出图直接复制粘贴，省去重复撰写 Prompt 的时间。
- **环节5 — 角色与场景设计**：使用即梦（https://jimeng.jianying.com/）生成角色形象，主角需生成：脸部特写 + 三视图（正/侧/背），女主、配角分别单独设计，群众演员无需设计；场景图同样用即梦生成，必须与人物保持同一画风。
- **环节6 — 出分镜图**：将分镜提示词输入即梦，参考已有角色图和场景图生成画面，不满意则重新抽卡，直到满意为止，保持人物和场景一致性。
- **环节7 — 出分镜视频**：在即梦中输入画面运动描述（如"阳光角度慢慢变化，风吹拂着衣服，画面左移"），生成对应视频片段；点击视频右下角"AI音效"按钮，自动为每段视频生成匹配音效，所有分镜均需执行此步骤。
- **环节8 — 配音**：使用 dubbingx（https://www.dubbingx.com/）输入角色台词，选择合适音色生成配音，全片同一角色必须保持音色一致。
- **环节9 — 剪辑合成**：使用剪映导入所有视频片段，按分镜顺序剪辑合成，搭配音效，保持音效与画面对应节奏。

### 🎯 关键洞察

**为什么分镜脚本是核心卡点**：
剧本是文字语言，AI 出图需要视觉语言。分镜脚本是两者之间的"翻译层"。没有分镜脚本直接出图，会导致画面风格混乱、镜头逻辑断裂。分镜表格中的"景别+摄法+画面内容"三列，直接决定了最终视频的专业度。

**为什么要先做角色三视图再出分镜图**：
即梦生成图片时，参考图（角色三视图）作为风格锚点，能大幅提升同一角色在不同镜头中的外貌一致性。跳过这步直接出图，角色脸型、发型、服装会在不同镜头间漂移，导致观众无法识别同一人物。

**情绪转变切镜原则的实际意义**：
原文举例"小明从开心变成愤怒"需拆成4个镜头（笑容满面→笑容僵住→表情阴沉→愤怒爆发），这不是繁琐，而是 AI 出图的物理限制——单张图片无法表达动态情绪变化，必须用多张图的序列来模拟。

### 🛠️ 操作流程

**1. 准备阶段**

- 准备小说片段（现代校园类，只需开头部分）
- 注册账号：DeepSeek、即梦、dubbingx、剪映

**2. 核心执行**

- Step 1：将小说片段 + 编剧Prompt → DeepSeek → 得到电影化剧本
- Step 2：将剧本 + 分镜导演Prompt → DeepSeek → 得到分镜表格（含AI提示词列）
- Step 3：即梦生成主要角色三视图（正/侧/背）+ 脸部特写，生成主要场景图
- Step 4：将分镜提示词逐条输入即梦，参考角色图和场景图出图，不满意抽卡
- Step 5：即梦将分镜图转视频，输入运动描述，点击AI音效自动生成
- Step 6：dubbingx 生成角色配音，保持音色一致

**3. 验证与优化**

- 检查同一角色在不同镜头中外貌是否一致（脸型、发型、服装）
- 检查场景图与人物图画风是否统一
- 剪映中检查音效与画面是否对应
- 整体节奏是否符合约1分钟/集的时长设定

### 📝 编剧 Prompt 完整模板

```
你是一个经验丰富的编剧，博览群书，擅长捕捉抖音、视频号、小红书、快手等自媒体平台用户喜好，参与过大量剧本创作，并且看过无数的电影、电视剧和动漫，精通各种类型微电影剧本撰写具备深厚的微电影剧本撰写知识和创作能力，能够创作优秀的剧本，根据你剧本创作的动漫和电视剧，总会大卖。

这是一个小说片段，学习它，请通读全文，我要改编成短剧，这一集大约1分钟，按照要求，并运用自身能力编写剧本，可在工作中使用动漫、微短剧、电视剧、电影等等知识库。

请按照如下格式来帮我写这一集剧本，请记住下面的格式：
{
1、【场景题头】: 格式为：场号 场景名 [内/外] [日/夜]，示例：1-1 废弃仓库 [内] [夜]（场景题头要加粗）
2、【画面/动作】: 所有环境描写、人物动作、镜头画面，必须以符号"△"开头。严禁出现心理描写，必须转化为可视化的动作或表情。
3、【对白】: 采用"紧凑格式"，人名后加冒号，人名要加粗。格式：角色名：(情绪/动作提示) 台词内容
4、【特殊音效与视觉】: 重要声音/音效用【音效】标注，屏幕文字/字幕用【字幕】标注。
5、【专业术语】: 画外音标注(O.S.)，内心独白标注(V.O.)。
6、每一集要在最前面把当集出场人物名称写出来，加粗。
7、不要把"【画面/动作】、【场景题头】、【对白】、【特殊音效与视觉】、【专业术语】"这些字眼放到剧本里。
8、每一句话之间都要换行。
9、要遵守剧情路线。
}

[在此粘贴小说片段]

### 📝 分镜导演 Prompt 完整模板

```
角色定义：
你是一个拥有20年行业资历的影视导演及分镜设计师。你的核心专业能力在于把文字剧本转换成视频的分镜脚本，熟练运用蒙太奇思维和视听语言。

分镜脚本拆分核心原则(必须遵守)：
- 情绪转变需切镜：每次显著情绪转变（平静→震惊→愤怒）获得独立分镜，情绪积累用2-3个镜头铺垫
- 复杂动作需切镜：按"动机/准备→执行过程→结果/反应"拆解，复杂动作细分3-5步（如"喝水"拆成：伸手→拿水→喝水→放下水瓶）
- 人物变化需切镜：人物数量增减、位置关系改变、主导-从属关系转换时启用新分镜
- 空间转场需切镜：场景切换必须设立新分镜编号，两空间之间设计1-2秒转场镜头
- 时间跳跃需切镜：非连续时间流逝必须通过独立分镜视觉化呈现
- 对话视线乒乓：说话者→聆听者→说话者切换，每次发言者更替触发镜头切换
- 摄影机变化：景别、角度、运动方式发生有意义改变时标志新镜头开始
- 视觉特效独占
