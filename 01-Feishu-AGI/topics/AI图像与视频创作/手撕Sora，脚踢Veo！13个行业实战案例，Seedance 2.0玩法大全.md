### 📋 核心分析

**战略价值**: Seedance 2.0 是首个具备"导演思维+世界知识+全模态参考"的视频生成模型，一句话提示词即可覆盖餐饮/电商/家装/影视等13个行业场景，且全部能力可通过 API 程序化调用，是视频内容生产自动化的基础设施级工具。

**核心逻辑**:

- **"写意图、不写细节"是核心提示词原则**：三个视频加起来不到30字提示词。有世界知识的场景（兰州拉面、无印良品、健身动作）直接写目标，分镜、镜头语言、节奏全部交给模型，比自己写更好。
- **模型自带世界知识**：知道兰州拉面的制作流程（自动安排揉面/拉面/浇汤/升格慢动作捞面）、MUJI品牌DNA（自选极简木质画面+写出"去除多余，回归本质"口播词+极简钢琴配乐）、健身动作目标肌群（准确识别背阔肌、主动提示"不要用惯性借力"错误）。
- **全模态参考上限**：单次支持 9张图片 + 3段视频 + 3段音频，可完全保留素材或只提取某些元素，参考智能到连分镜图上的文字都不会出现在生成视频里。
- **UI设计稿→3D宣传片需两步走**：先用 Nano Banana Pro 或 Seedream 图像模型将截图处理成 Fluent/Apple 风格渲染图，再喂给 Seedance 2.0 生成动效视频。单张图片抽卡效果更好，多张图片可一键带字幕口播。
- **真人动作迁移的关键词**：人物相对摄影机静止（世界运动）的场景需加 `CAMERA MOUNTED ON JOHN`、`LOCKED-ON SHOT`、`FIXED-TO-ACTOR` 三个关键词，否则动作一致性无法保持。
- **音频参考可修改情绪**：上传音频后提示词写"表现得更激动"，模型真的能调整音频情绪表现，不只是原样播放。单独上传音频不行时，用一张透明/纯白 PNG 图片作为占位图绕过限制。
- **小说原文直接生成动画**：不需要先写分镜脚本，直接粘贴小说文本+参考视频（如《雾山五行》片段），模型自动对齐故事情节和画风。续拍方式：`将@视频1延长15s，具体信息为：[后续小说文本]`，理论上可无限延长。
- **跨镜头一致性三要素**：图像参考保证人像一致性，音频参考保证声音一致性，视频参考保证动作/风格一致性，三者可同时叠加使用。
- **API 上线后可构建三类 Agent 管线**：①产品宣发 Agent（读取更新→截图→生成视频）；②口播视频 Agent（热点→文案→音频→场景图→多段视频→拼接长视频）；③电商/家装/小说转漫剧批量生产 Agent。
- **家装视频需两步走**：先用图像模型从户型图生成九宫格分镜图（含空间顺序和风格描述），再将户型图+分镜图同时发给 Seedance 2.0，模型会按分镜顺序生成沉浸式参观视频。

### 🎯 关键洞察

**为什么 Seedance 2.0 是"视频领域的 ChatGPT 时刻"**：

以往视频模型是"你说什么它画什么"的像素渲染器，没有语义理解，没有行业知识，没有叙事逻辑。Seedance 2.0 的突破在于：

- 原因：模型内化了大量世界知识和影视语言，能理解"兰州拉面广告"背后的拍摄惯例，能理解"MUJI"背后的品牌哲学。
- 动作：用户只需提供意图（"生成兰州拉面广告"），模型自动完成导演工作（选择升格摄影、安排分镜顺序、选配乐）。
- 结果：非专业用户的一句话提示词，产出效果等同于有经验的视频导演操作。

**音频情绪修改能力的意义**：这说明 Seedance 2.0 不只是"素材拼接器"，而是对输入素材有理解和再创作能力，这是构建自动化 Agent 的关键前提——Agent 可以通过提示词动态调整输出情绪和风格，而不是只能原样输出。

### 🛠️ 操作流程

**UI设计稿→微软Fluent风宣传片（完整SOP）**

1. 准备阶段：截取产品 UI 截图（原始截图即可，无需美化）

2. 图片处理（Nano Banana Pro 或 Seedream）：
```
为这张 UI 设计稿添加类似于微软 UI 宣传片的包装，不要丢失原有内容，具体要求为一种"数字材质感"。
光感 (Light): 利用点光源、边缘光（Rim Light）勾勒物体轮廓，制造焦点。
材质 (Material): 大量使用亚克力 (Acrylic) 材质——带有磨砂玻璃质感的半透明效果，以及类似陶土、金属和硅胶的物理材质感。
深度 (Depth): 强调 Z 轴概念，UI 元素分层，通过阴影和视差滚动体现层级关系。
微距美学: 镜头贴近，看到图标边缘倒角、材质表面细微颗粒（Noise）。
CMF 设计: 每个图标仿佛精心打磨的工艺品，颜色有厚度、有光泽。
光线追踪渲染: 半透明材质透出漫射光，营造高级温润氛围。
```

3. 单张图片生成视频（Seedance 2.0）：
```
基于 @图片1 帮我生成一个类似微软 Fluent UI 宣传片的亚克力和玻璃质感动效设计视频，
用丝滑的动效设计和不同角度的运镜展示 UI 细节，最后无缝过渡到产品名称"[产品名]"
```

4. 多张图片生成视频（Seedance 2.0）：
```
帮我将 @图片6 @图片5 @图片4 @图片3 @图片2 @图片1 这几个 UI 展示图，
细节展示和 @图片7 icon 变成多分镜多角度的应用宣传片，并搭配合适的口播介绍，
为其增加丰富的转场效果和细节展示，每个分镜的转换需要连贯顺畅。
产品介绍为：[产品介绍文本]
宣传视频制作思路为：用电影级的 CGI 微距视角，将虚拟界面重塑为精密打磨的实体工业品。
通过极细腻的材质渲染（亚克力与玻璃质感）和光影深度，让图标呈现可触摸的真实感；
结合符合物理惯性的动态设计与 ASMR 级触感音效，赋予像素以重量、弹性与节奏。
类似 Microsoft 广告风格，增加玻璃风格动画。
```

**户型图→样板间参观视频（完整SOP）**

1. 第一步，用图像模型生成九宫格分镜：
```
基于当前的户型图，为我设计一套九宫格的样板间参观视频分镜图。
分镜图需要严格遵循以下顺序，所有的照片都从门口的玄关进入，依托于这个坐标点拍摄。
从玄关看向右手的厨房，看向左手的卫生间，再到客厅，最后到主卧和次卧。
空间关系要跟户型图对应，风格为温馨的北欧风格。
```

2. 第二步，将户型图+分镜图发给 Seedance 2.0：
```
参考 @图片2设计稿分镜 和 @图片1户型图，生成一段沉浸式的样板间参观视频，
严格按照 @图片2 的分镜顺序生成视频，模拟真实看房的行走路线，
可以切镜头，注意自然光线变化和材质质感。
```

**小说→动画续拍（完整SOP）**

1. 首次生成：
```
帮我根据下面的小说剧情要求生成动漫，将画风动作打斗表现严格对齐 @视频1 的风格。
剧情要求为：[粘贴小说原文]
```

2. 续拍延长：
```
将 @视频1 延长15s，具体信息为：[后续小说文本]
```

### 📝 避坑指南

- ⚠️ **真人主体参考限制**：目前限制了真人主体参考，必须经过本人验证或取得授权，服装换装等场景需注意合规。
- ⚠️ **Vlog风格迁移必须描述风格特色**：如果提示词不写参考Vlog的具体特色（硬切/ASMR拟音/3D文字/卡点），模型学习效果会明显变差；色彩表现、环境音、小动效目前仍是弱项。
- ⚠️ **人物相对摄影机静止的场景必须加三个关键词**：`CAMERA MOUNTED ON JOHN`、`LOCKED-ON SHOT`、`FIXED-TO-ACTOR`，否则动作一致性无法保持（骑扫帚飞翔等场景）。
- ⚠️ **logo细节还原受分辨率限制**：服装上的logo在低分辨率或原图模糊时还原效果差，属于当前版本硬限制。
- ⚠️ **样板间空间方向偶有偏差**：如卫生间应左转但模型右转，多抽几次卡可解决，不是必现问题。
- ⚠️ **不支持单独上传音频时**：用一张透明或纯白的 PNG 图片作为占位图，绕过限制上传音频。
- ⚠️ **UI宣传片不能省略图片处理步骤**：直接把原始截图发给 Seedance 2.0 效果差，必须先经过图像模型处理成 Fluent/Apple 风格渲染图再生成视频。
- ⚠️ **API 时间节点**：Seedance 2.0 API 在春节后上线火山引擎，支持全模态输入，体验地址：`exp.volcengine.com/ark/vision`。
