### 📋 核心分析

**战略价值**: 用「即梦AI」三步流水线（AI生图 → 合成海报 → 图生视频）零设计基础制作带丝滑动效的春节动态海报，核心依赖 Seedance 3.5 Pro 模型的微动效处理能力。

**核心逻辑**:
- **整体流程只有3步**：生成毛绒元素图 → 合成静态海报 → 生成动态视频，全程在即梦AI内完成，无需设计软件
- **元素生成用白底图**：输入素材最好是白底，生成的3D毛绒元素边缘干净，后续合成时不会有杂色背景干扰
- **Prompt精度决定合成质量**：合成海报时提示词固定为「把图一放到图二中间的位置，保持图一的形象不变，让角色自然融于图片」，顺序不能乱（图一=元素，图二=背景）
- **海报尺寸标准**：背景图建议 1080×1920（竖版），适配手机屏幕和朋友圈分享
- **动效用"首尾帧"功能实现**：空白背景作首帧，含元素的海报作尾帧，AI自动补间生成丝滑转场，不需要手动做关键帧
- **视频模型选 3.5 Pro**：生成效果更稳定，输出自带音效的5秒视频；代价是消耗积分较多，需权衡
- **多段视频连接靠"首尾帧"串联**：上一张海报作首帧，下一张作尾帧，依次生成过渡片段，实现多段无缝衔接
- **最终剪辑极简**：4条视频拖入剪映按顺序拼接导出，不需要剪辑技巧，不需要配音
- **可扩展性强**：同一套流程替换元素即可复用于情人节、元宵节等任意节日，或将真人照片生成3D毛绒形象作为海报主角

### 🛠️ 操作流程

**第一步：生成毛绒风格元素**
1. 准备素材图（锦鲤、小马、红包等），优先选白底图
2. 打开「即梦AI」→ 图片生成
3. 上传素材图，输入提示词：
   ```
   把图片转为3D毛绒材质
   ```
4. 挑选效果最好的一张下载备用

**第二步：合成静态海报**
1. 准备背景图，尺寸 **1080×1920**（竖版）
2. 即梦AI → 图片生成，依次上传：图一（毛绒元素）+ 图二（海报背景）
3. 输入提示词：
   ```
   把图一放到图二中间的位置，保持图一的形象不变，让角色自然融于图片
   ```
4. 挑选满意版本下载，重复此步骤完成所有海报（如4张）

**第三步：让海报动起来**
1. 即梦AI → 视频生成 → 开启「首尾帧」功能
2. 首帧：空白海报背景；尾帧：含元素的完整海报
3. 输入提示词：
   ```
   添加丝滑有趣的转场，固定镜头
   ```
4. 模型选择：**视频3.5 Pro（Seedance 3.5 Pro）**，输出5秒带音效视频
5. 串联多段：上一张海报作首帧 → 下一张海报作尾帧，重复生成过渡片段
6. 将全部视频片段拖入**剪映**，按顺序排列，直接导出

|----------|-------------|---------|-----------|

### 📝 避坑指南

- ⚠️ 元素图必须白底，彩色背景会导致合成时出现杂色边缘，难以自然融合
- ⚠️ 合成提示词中图一/图二顺序严格对应上传顺序，搞反会导致元素和背景互换
- ⚠️ 视频3.5 Pro积分消耗高，批量生成前先用单张测试效果再批量跑
- ⚠️ 首尾帧串联时，相邻两段视频的首/尾帧必须是同一张海报，否则转场会跳帧不连贯
