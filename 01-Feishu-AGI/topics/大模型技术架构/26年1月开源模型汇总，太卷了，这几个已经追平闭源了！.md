### 📋 核心分析

**战略价值**: 2026年1月开源模型爆发式涌现，覆盖多模态、语音、OCR、医疗、机器人控制等全赛道，部分10B级模型已追平甚至超越闭源大模型。

**核心逻辑**:

- **资本加速**：智谱、MiniMax上市，Kimi融资5亿美元，阶跃融资50亿人民币，资金驱动开源提速，26年定性为"大模型应用年"。
- **DeepSeek架构预判**：1月发布 mHC 和 Engram 两篇论文，基本锁定 DeepSeek-V4 架构方向，2月预计落地验证。
- **Qwen全栈覆盖**：Image、ASR（0.6B/1.7B）、TTS（0.6B/1.7B）、Embedding、Reranker（2B/8B）、Max-Thinking 全线开源，标志 Qwen3 时代收尾，Qwen4 预期临近。
- **小参数高性价比成主流**：PaddleOCR-VL 1.5（0.9B）、AgentCPM-Explore（4B）、STEP3-VL-10B（媲美规模大10-20倍的开源模型）均验证"小而精"路线可行。
- **MoE架构普及**：GLM-4.7-Flash（30B-A3B）、LongCat-Flash-Lite（68.5B参数，激活仅2.9B~4.5B）均采用MoE，推理成本大幅压缩。
- **医疗垂直赛道加速**：蚂蚁+浙江卫健 AntAngelMed（100B总参/6.1B激活）、百川 M3（235B）双双开源，主打降低幻觉。
- **Computer Use 落地**：美团 EvoCUA-32B（1月5日）+ 8B版本（1月13日）开源，国内首批 Computer Use 专项模型。
- **前端/Agent 能力分化**：Kimi K2.5 前端效果被评为"国内无敌，直逼 Gemini 3.0 Pro"；但整体国内开源模型 Agent 使用 skills 的稳定性仍不足，是明确短板。
- **机器人控制模型元年**：蚂蚁 LingBot-VA（因果视频-动作世界模型）、宇树 UnifoLM-VLA-0（基于 Qwen2.5-VL-7B 持续预训练）同日开源，VLA 赛道正式起跑。
- **视频生成多路并进**：快手 UniVideo（理解+编辑+生成三合一）、腾讯 HunyuanImage-3.0-Instruct（图生图+提示词增强）、Skywork SkyReels-V3（图转视频+音频驱动+视频延长）三家同月落地。

### 🎯 关键洞察

**Kimi K2.5 为何前端能力突出**：原生多模态架构（非拼接），同时支持视觉+文本输入、思考/非思考双模式、对话/Agent 双任务，架构统一性带来前端渲染和交互的一致性优势，这是拼接式多模态模型的结构性短板。

**DeepSeek Engram 的实际意义**：不是通用记忆模块，而是"条件记忆"——外挂词典形式，针对局部知识快速查表，解决的是模型对特定领域知识理解滞后的问题，可理解为轻量级 RAG 的架构内化版本。

**Agent 稳定性是国内开源的天花板**：作者明确指出，从 skills 使用稳定性判断，国内开源 Agent 水平整体不达标，这意味着当前开源模型在工具调用、多步规划的可靠性上仍有系统性缺口。

### 💡 具体案例/数据

- **STEP3-VL-10B**：10B参数媲美甚至超越规模大10-20倍的开源模型，即等效对标100B~200B量级。
- **LongCat-Flash-Lite**：总参数68.5B，但激活参数仅2.9B~4.5B，MoE稀疏激活率约4%~6.5%，同时集成N-gram嵌入表（创新点）。
- **AntAngelMed**：100B总参，激活6.1B，基于Ling-flash-2.0增量训练，联合浙江卫健委数据。
- **PaddleOCR-VL 1.5**：0.9B参数，作者评为"性价比之王"，并与DeepSeek-OCR-2做了全面实测对比。
- **Kimi K2.5**：前端能力被评为"国内无敌，直逼Gemini 3.0 Pro"，是当月最高评价。

### 📝 避坑指南

- ⚠️ **Agent 稳定性不可高估**：国内开源模型在 skills 调用稳定性上存在系统性缺口，生产环境 Agent 任务慎用，需做 fallback 机制。
- ⚠️ **MoE 模型部署注意激活参数 vs 总参数**：GLM-4.7-Flash（30B总/3B激活）、LongCat-Flash-Lite（68.5B总/2.9B~4.5B激活），显存需求按激活参数估算，但加载时需容纳全量参数。
- ⚠️ **Z-Image 标准版是非蒸馏完整模型**：区别于蒸馏版，推理成本更高，部署前确认资源预算。
- ⚠️ **DeepSeek Engram 是外挂模块非端到端训练**：使用时需额外维护词典文件，知识更新需同步更新词典，不是 fine-tune 替代品。
- ⚠️ **2月将更卷**：GLM、Qwen、DeepSeek 均有大版本预告，当前选型建议等2月版本落地后再做技术栈锁定。
