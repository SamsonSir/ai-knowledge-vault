### 📋 核心分析

**战略价值**: 用 VSCode + Cline + Magic UI MCP Server + 无问芯穹 GenStudio API，2-3分钟零人工干预生成完整科技企业官网 HTML。

**核心逻辑**:

- **工具链组合**: VSCode（IDE）+ Cline（AI编码扩展）+ Magic UI MCP Server（UI组件上下文服务）+ GenStudio API（推理后端），四者缺一不可
- **MCP 的角色**: Magic UI MCP Server 基于开放协议 MCP，标准化向 LLM 提供 Magic UI 组件上下文，让 AI IDE 直接访问并调用组件，降低代码生成错误率
- **API 兼容性**: GenStudio 提供 OpenAI Compatible 接口，Cline 可直接对接，无需额外适配层
- **模型选择**: 使用 `deepseek-v3` 作为 Act 功能模型；QWQ-32B 等模型支持 function call，适合 MCP 工具调用场景
- **Plan/Act 独立配置**: Cline 的 Plan 和 Act 功能 API 配置相互独立，必须单独为 Act 配置 API，否则执行阶段无法调用
- **Node.js 是 MCP 前置依赖**: MCP Server 本质是运行在本地的 Node.js 程序，未安装 Node.js 则 MCP Server 无法启动
- **Mac 用户需额外安装 Homebrew**: 通过国内镜像脚本安装，避免官方源网络问题
- **Magic UI MCP 安装需 GitHub 授权**: 安装过程需登录 GitHub 并获取 Magic UI MCP Server 密钥，安装耗时约 5-6 分钟
- **Prompt 设计决定输出质量**: 需在 Prompt 中明确指定设计风格（深色主色调、蓝色+银色点缀、科技感+专业感），否则生成结果风格随机
- **输出物为完整 HTML 文件**: 生成结果是可直接打开的单 HTML 文件，包含公司简介、产品展示、联系方式等完整页面结构

### 🛠️ 操作流程

**1. 准备阶段：环境配置**

检查 Node.js 是否已安装：
```bash
node -v
```
未安装则前往：`https://nodejs.org/zh-cn/download` 下载安装。

Mac 用户安装 Homebrew（使用国内镜像）：
```bash
/bin/bash -c "$(curl -fsSL https://gitee.com/cunkai/HomebrewCN/raw/master/Homebrew.sh)"
```

**2. 核心执行：配置 Cline + GenStudio API**

- VSCode 安装 Cline 扩展
- 点击 Cline 图标 → 打开设置面板
- ⚠️ 注意：必须配置 **Act 功能** 的 API（Plan 和 Act 独立配置）
- API Provider 选择：`OpenAI Compatible`
- 填写以下参数：

| 参数 | 值 |
|------|-----|
| Base URL | `https://cloud.infini-ai.com/maas/v1` |
| API Key | GenStudio 控制台获取的 API 密钥 |
| Model ID | `deepseek-v3` |

**3. 安装 Magic UI MCP Server**

- 在 Cline 中打开 MCP Marketplace
- 搜索 `Magic UI` → 点击 Install
- 按提示完成 GitHub 授权登录
- 输入 Magic UI MCP Server 密钥
- 等待安装完成（约 5-6 分钟）

**4. 实战执行：生成官网首页**

在 Cline 对话框输入以下 Prompt：

```
使用Magic MCP创建一个科技企业的官网首页，页面采用简洁现代的设计风格，
整体以深色为主色调，搭配蓝色和银色点缀，突出科技感和专业感。
```

- Magic UI 自动生成 HTML 代码（约 2-3 分钟）
- 生成完成后获得完整 HTML 文件路径
- 直接用浏览器打开该文件即可预览

**5. 验证与优化**

- 检查页面是否包含：公司简介、产品展示、联系方式
- 如需调整风格，修改 Prompt 中的颜色/风格描述后重新生成

### 📝 避坑指南

- ⚠️ **Plan/Act 配置不要混淆**：Cline 的 Plan 功能和 Act 功能 API 配置完全独立，生成代码需配置 Act 的 API，漏配会导致执行阶段报错
- ⚠️ **Node.js 必须先装**：MCP Server 是本地 Node.js 进程，跳过此步骤 Magic UI MCP 安装后无法运行
- ⚠️ **免费版有频率限制**：基础免费版 API 调用频率受限，复杂页面生成可能因超频中断，需申请提频或等待升级版
- ⚠️ **Magic UI 安装需 GitHub 账号**：安装过程强制 GitHub 授权，无账号需提前注册
- ⚠️ **Prompt 必须包含风格描述**：不指定颜色、风格、氛围，生成结果质量和一致性无法保证
