### 📋 核心分析

**战略价值**: OpenAI 官方社区整理的销售岗位 25 个即插即用 Prompt，覆盖五大模块，解决销售日常 80% 的重复性文字工作。

**核心逻辑**:

- 销售岗位的 AI 使用障碍不是能力问题，是"不知道怎么问"——解法是用岗位场景强绑定的 Prompt 模板，而非通用提问
- 原始资源来自 OpenAI 官方社区：https://academy.openai.com/home/clubs/work-users-ynjqu/resources/use-cases-sales
- 25 个 Prompt 分五大模块：对外联络与沟通（5个）、销售策略与规划（5个）、竞争情报与赋能（5个）、数据分析与绩效洞察（5个）、视觉素材和销售材料（5个）
- 策略类 Prompt（模块二）建议使用推理模型（如 o1/o3）以获得更深层战略洞察
- 竞争情报类 Prompt（模块三）建议开启深度研究或网络搜索功能，获取实时数据
- 数据分析类 Prompt（模块四）需配合上传 CSV 文件使用，AI 直接计算转化率、成交率、交易速度
- 有效 Prompt 的三要素：①描述清楚需求（具体到行业+职位+场景）②明确目标与痛点（提高打开率 or 化解异议）③提供上下文（产品价值主张+客户处境）
- AI 放大的是你已有的工作结构，没有方法论的人用 AI 只会得到泛泛建议——先拆解岗位任务，再把判断标准交给 AI
- 这 25 个 Prompt 可封装成个人专属 Sales Skill，遇到对应场景直接调用，无需每次重新构思
- 适配国内模型（豆包、Kimi、千问）与 ChatGPT 底层逻辑一致，模板可直接迁移使用

### 🛠️ 操作流程

**五大模块 · 25 个完整 Prompt**

---

#### 一、对外联络与沟通

**1. 撰写个性化陌生开发邮件**
```
请撰写一封简短而引人入胜的陌生开发邮件，发送给【公司名称】的【职位名称】，
介绍我们的产品。背景：【插入产品价值主张或理想客户画像信息】。
请将邮件内容格式化为适合电子邮件阅读的文本格式。
```

**2. 重新制作演示后续邮件**
```
演示结束后，请修改这封后续邮件，使其更具咨询性。
原文：【粘贴至此处】。
邮件内容应包含内容回顾、后续步骤以及预约通话的行动号召。
```

**3. 为重要客户拟定续约方案**
```
根据【客户名称】的续约历史和价值数据（[粘贴数据]），撰写一份续约方案。
方案中需包含关键的投资回报率证明和续约建议。
```

**4. 创建代表活动摘要**
```
撰写每日工作总结，概述销售代表的关键活动。
输入内容：[粘贴通话记录摘要或 CRM 导出文件]。
内容应积极向上、简洁明了。
```

**5. 关于最新进展草案**
```
请向高管总结本月我们的项目储备情况。
输入内容：[粘贴数据]。
包括项目储备总量、主要风险、最大成果以及预测信心。
```

---

#### 二、销售策略与规划

**6. 制定战略客户计划**
```
为[客户名称]创建账户计划。
输入：公司简介、已知优先级、当前产品使用情况、利益相关者和续约日期。
输出结构化计划，包含目标、风险、机遇。
```

**7. 设计区域规划框架**
```
请为下一财年制定区域规划指南。
所需信息：团队人数、目标行业、地区和历史收入。
推荐分配方法。
```

**8. 利用公司概况数据确定账户优先级**
```
我有以下账户列表：[粘贴示例]。
请根据[标准：行业、规模、融资、技术栈]对它们进行优先级排序。
```

**9. 利用加权评分发现高潜力账户**
```
根据[插入规则——例如，公司规模、互动评分]对账户进行评分。
数据：[上传列表]。
输出前 10 名。
```

**10. 区域市场进入规划**
```
我正在评估我们的[产品]进入[地区]市场的可行性。
研究当地购买行为、竞争、经济和监管。
形成可行性评估。
```

---

#### 三、竞争情报与赋能

**11. 为竞争对手制作对战卡**
```
为[竞争对手名称]创建竞争分析卡。
信息：[插入定位数据]。
内容包括优势、劣势、制胜策略和简短发言稿。
```

**12. 竞争定位分析**
```
针对[竞争对手名称]准备报告。
研究其定价模式、近期客户流失情况和销售策略。
对比我们的优势。
```

**13. 创建一份销售赋能单页文件**
```
制作一份单页宣传资料，向[目标客户画像]推介[产品名称]。
包括关键优势、功能、应用场景。
```

**14. 准备销售异议反驳策略**
```
针对以下常见反对意见，撰写反驳文：[插入 2-3 个反对意见]。
措辞应自然自信，提供佐证数据。
```

**15. 在公共领域寻找客户证明点**
```
研究近期关于[我们的产品或竞品]的在线评论、社交媒体提及。
总结排名前五的评价。
```

---

#### 四、数据分析与绩效洞察

**16. 按阶段分析流程转化率**
```
分析此销售流程导出文件。
计算各阶段转化率，找出最大流失点。
数据：[上传 CSV]。
```

**17. 按成交率识别业绩最佳销售代表**
```
根据活动和已成交数据，计算成交率并排序。
输出排名及优势描述。
```

**18. 可视化各季度的交易速度**
```
计算每个季度的平均交易速度（线索到成交天数）。
数据：[上传]。
```

**19. 总结营销活动对已成交交易的归因**
```
将营销活动来源与已成交订单匹配。
确定哪个活动带来的收入最多。
```

**20. 生成性能对比图表**
```
比较业绩最佳和最差的代表。
绘制趋势图并突出关键差异。
```

---

#### 五、视觉素材和销售材料

**21. 可视化销售流程漏斗**
```
创建一个漏斗图，展示阶段：[插入阶段名称]。
确保简洁易读。
```

**22. 可视化 B2B 销售漏斗**
```
创建标准 B2B SaaS 销售漏斗图，包含寻找线索、演示、提案、成交。
```

**23. 绘制关键销售人物图**
```
为以下角色创作专业插图：(1)CFO，(2)IT 副总裁，(3)运营经理。
风格简洁现代。
```

**24. 创建区域覆盖范围图**
```
制作一张简化的地图，按颜色划分销售区域：西部、中部、东部。
```

**25. 设计一个团队庆祝图形**
```
设计一张图片，用于表彰"月度最佳员工"。
包含姓名照片占位符和奖杯。
```

### 📦 配置/工具详表

| 模块 | 推荐模型/功能 | 核心输入 | 输出形式 |
|------|------------|---------|---------|
| 对外联络与沟通 | 标准对话模型 | 客户名称、职位、产品价值主张、CRM导出 | 邮件正文、方案文档 |
| 销售策略与规划 | 推理模型（o1/o3） | 客户简介、团队规模、历史收入、账户列表 | 结构化计划、优先级排名 |
| 竞争情报与赋能 | 开启网络搜索/深度研究 | 竞品名称、定位数据、反对意见 | 对战卡、单页文件、反驳话术 |
| 数据分析与绩效洞察 | 标准模型 + 文件上传 | CSV 销售数据、活动归因数据 | 转化率表、排名、趋势图 |
| 视觉素材和销售材料 | 图像生成模型（DALL·E等） | 阶段名称、区域划分、人物角色 | 漏斗图、地图、插图 |

### 💡 Prompt 三要素框架

| 要素 | 错误示范 | 正确示范 |
|------|---------|---------|
| 描述清楚需求 | "帮我写个邮件" | "帮我给SaaS行业的CEO写一封合作开发邮件" |
| 明确目标与痛点 | "写得好一点" | "目标是提高邮件打开率 / 化解客户的价格异议" |
| 提供上下文 | 无背景直接提问 | "我们产品的核心价值是X，客户目前面临Y问题" |

---

### 📝 避坑指南

- ⚠️ 模块二（策略规划）用普通对话模型效果有限，务必切换到推理模型（o1/o3/深度思考模式）
- ⚠️ 模块三（竞争情报）中 Prompt 15 需要联网，离线模型无法获取实时评论数据
- ⚠️ 模块四（数据分析）所有 Prompt 依赖真实数据上传，占位符 `[上传 CSV]` 不能省略，否则 AI 只能给出模板而非实际计算结果
- ⚠️ 模块五（视觉素材）需要支持图像生成的模型，纯文本模型只能输出描述，无法直接生成图片
- ⚠️ 这 25 个 Prompt 是模板骨架，`【】` 和 `[]` 内的占位符必须替换为真实业务数据，否则输出质量大幅下降
